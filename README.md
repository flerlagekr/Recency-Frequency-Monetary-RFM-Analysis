# RFM Analysis
Python code and Tableau Prep flows for performing RFM analysis on a list of donations. Currently includes two versions of each. One does a traditional RFM encoding with scores of buckets 1-5 and combined three-digit RFM score. The second creates scores of 0-10 (not buckets) with a combined score that averages the 3. This allows for more nuance and better quantitative analysis.
